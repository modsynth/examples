# Fullstack Example

Complete full-stack application using all Modsynth modules.

## Architecture

- Backend: Go (all backend modules)
- Frontend: React (all frontend modules)
- Database: PostgreSQL
- Cache: Redis
- Monitoring: Prometheus + Grafana

## Setup

```bash
docker-compose up
```

- Frontend: http://localhost:3000
- Backend: http://localhost:8080
- Grafana: http://localhost:3001

## Features

- User authentication
- CRUD operations
- Real-time updates (WebSocket)
- Monitoring dashboard
- Multi-language support
- Data visualization
